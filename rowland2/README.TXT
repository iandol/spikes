
Make sure the visual basic 6 runtime dll files are installed on the system.

Make sure matlab release 11 or more recent is being used.

Copy the contents of the rsmatlab directory to a directory on the machine on which the
routines will be used, name this directory whatever you want.

To run the program...
1. Start matlab.
2. type 'cd ' and then the path and directory in which the routines have been put

[eg.  'cd d:\matlabroutines'] etc.

3. type 'threedplotter'
